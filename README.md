# 役に立たなかった記録アプリ

## はじめに

このアプリは、「役に立たなかったこと」を記録するための、実用性を目的としないミニアプリです。

何かを達成した記録や、前向きな成果ではなく、  
時間を使ったものの特に意味がなかった出来事や、  
振り返るほどでもない体験を、そのまま残すことを想定しています。

記録する内容に価値があるかどうかは問いません。  
役に立たなかったことを、役に立たせようとしないことを前提にしています。

機能は最小限です。  
入力して、保存して、一覧で眺めるだけです。

---

## 終わりに

このアプリは、生産性や改善を促すためのものではありません。  
記録したからといって、何かが良くなることも、悪くなることもありません。

ただ、書いたという事実だけが残ります。

役に立たなかった時間も、確かにそこにあった時間として、  
淡々と置いておくための場所です。

必要がなくなったら、使わなくても構いません。  
続けなくても、完成度を高めなくても問題ありません。

このアプリ自体も、役に立たなくて大丈夫です。
